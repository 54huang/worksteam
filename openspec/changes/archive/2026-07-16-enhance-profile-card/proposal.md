## Why

当前个人名片页面布局单一（纯居中堆叠），缺少深色/浅色模式切换，信息展示不够丰富。需要增强视觉效果、改进桌面端空间利用、添加技能标签和职位信息，提升整体专业感。

## What Changes

- **布局重构**：桌面端头像左对齐，姓名和职位在头像右侧；移动端自动切换为居中堆叠
- **新增主题切换**：右上角太阳/月亮图标按钮，点击切换深色/浅色模式，`localStorage` 记住偏好
- **新增职位**：名字下方显示灰色职位文字（如"前端工程师"）
- **简介优化**：将原有长段个人简介替换为简短 Tagline（如"写代码十年，热爱开源"）
- **新增技能标签**：最多 8 个胶囊形技能标签，JS 数组渲染，`flex-wrap` 自动换行
- **联系信息区**：邮箱和 GitHub 移至卡片底部，用分割线隔开，桌面端左右分布
- **头像 Fallback**：保持现有的 SVG 占位图逻辑，无需改动

## Capabilities

### New Capabilities
- `theme-toggle`: 页面级深色/浅色主题切换，包含右上角切换按钮、CSS 变量双主题配色、localStorage 持久化

### Modified Capabilities
- `profile-card`: 布局改为桌面端左右结构（头像左 + 信息右）→ 移动端堆叠；新增职位、技能标签；简介改为短句；联系区移至底部加分割线

## Impact

- 修改 `index.html`：调整卡片 DOM 结构，新增主题切换按钮、职位、技能标签容器、分割线
- 修改 `styles.css`：新增深色主题 CSS 变量、重写卡片布局、技能标签样式、主题切换按钮样式、响应式断点调整
- 修改 `main.js`：新增主题切换交互逻辑、技能标签动态渲染
